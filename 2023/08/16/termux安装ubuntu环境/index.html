termux安装ubuntu环境

2023/8/16 原创 termux

termux为什么要安装ubuntu？

众所周知，Termux是Android平台上的一个模拟终端应用程序，它提供了许多常用的Linux命令和工具。尽管Termux本身已经提供了一些常用的Linux命令和工具，但有些情况下用户可能需要使用更多的工具和功能，而这些工具和功能在Android平台上并不直接可用。
在这种情况下，用户可以选择在Termux中安装Ubuntu，以获得更多的Linux命令、工具和功能。Ubuntu是一个广泛使用的Linux发行版，拥有庞大的软件包集合，包括各种开发工具、服务器软件、桌面环境等。
通过在Termux中安装Ubuntu，用户可以获得更多的Linux软件包，并且可以享受到更多的Linux操作系统的功能和特性。例如，使用Ubuntu，用户可以更方便地进行软件开发、运行服务器、使用桌面环境等。
这期文章，将在termux中讲述如何安装ubuntu。

准备工作

准备工具

1.termux APK

换源准备

termux在安装ubuntu或其他依赖包时，可能会遇到卡顿，慢或下载失败，所以建议换国内源，换源看这期:《termux换源》

开始教程

1.打开termux
2.输入安装脚本

bash -c "$(curl https://gitee.com/sansjtw/vnckali/raw/master/kalisansjtw.sh)"

然后回车
3.回车后会出现 “是否已安装dialog和fortune软件包？(y/n)” ，直接输入”n”回车，会下载两个软件包，下载中可能会停止，回车即可
4.下载完成后，会出现一个图形界面的”选择菜单”

我们需要termux下方的↑→↓←的键盘进行操作

通过↓选中”下载ubuntu”然后回车
5.可能还会弹出提示与”按回车键继续”，回车即可
6.此时会下载一些软件包等文件，我们只需要静静等待即可。
7.部分地方可能会下载很慢，那需要翻墙，比如这里

8.如果出现了”root@localhost:~#”，说明ubuntu安装成功了

初始的ubuntu没有多少程序软件包，通过

apt install <name>

进行安装
9.安装完成后，如果要在termux启动ubuntu，可以在脚本中打开，也可以用命令 

proot-distro login ubuntu

来代替此脚本进入Ubuntu
